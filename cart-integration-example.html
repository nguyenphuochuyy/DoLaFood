    Các thao tác với giỏ hàng:

    Hướng dẫn tích hợp:

    1. Include Cart CSS và JS: 
      <link rel="stylesheet" href="assets/css/tintuc.css">
<script src="assets/js/cart.js"></script>
<script src="assets/js/main.js"></script>
    2. Sử dụng Cart Template:

      Copy HTML từ file cart-template.html và paste vào header của trang.

    3. Thêm sản phẩm vào giỏ hàng: 
      // Sử dụng API static
ShoppingCart.addToCart({
    id: 'product_1',
    name: 'Tên sản phẩm',
    price: '99.000đ',
    unitPrice: 99000,
    quantity: 1,
    option: '3 miếng',
    notes: 'Ghi chú của khách hàng',
    image: 'assets/images/product.jpg'
});

// Hoặc sử dụng instance toàn cục
window.cart.addItem(productData);
    4. Lấy thông tin giỏ hàng: 
      const cartInfo = ShoppingCart.getCartInfo();
console.log('Tổng sản phẩm:', cartInfo.totalItems);
console.log('Tổng tiền:', cartInfo.totalPrice);
